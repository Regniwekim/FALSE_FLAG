1. PROJECT OVERVIEW & OBJECTIVES

FALSE_FLAG is a faithful, browser-based, real-time multiplayer adaptation of the classic Guess Who? board game. Players identify their opponent's secret country flag by asking yes/no questions in a live chat, eliminating flags on their personal board, and guessing correctly before their opponent does.

Strict rule adherence (from official Guess Who! rules):

  • Each player receives one unique random secret flag at game start
  • Each player has their own identical 24-flag board
  • Turns alternate: ask one yes/no question -> opponent answers truthfully -> asker eliminates on THEIR board only
  • Guess only on your turn; incorrect guess = instant loss
  • Chat is the only communication channel (yes/no questions only)
  • Support for Championship Play (best of 5)

Target: Fully playable in under 60 seconds per round, perfect for quick browser sessions.

3. SYSTEM ARCHITECTURE

  • Client: React SPA - one page per player with their own board, secret slot, chat, and controls
  • Server: Express + Socket.io server managing game rooms
  • Game Room: One room = two players + private game state (P1 secret, P2 secret, turn, eliminated sets, chat log)
  • Real-time sync: Server broadcasts only the information each player is allowed to see (never the opponent's secret)

4. SOCKET.IO EVENT SPECIFICATION (CRITICAL)

Connection & Lobby

  • create-room -> server returns room code
  • join-room (roomCode) -> both players get "game-started"

Core Game Events (server validates everything)

EventDirectionPayloadServer Action
ask-questionClient -> Server{ question: string }Validate it ends with "?", broadcast to opponent as "incoming-question"
answer-questionClient -> Server{ answer: "yes" | "no" }Broadcast answer to asker; advance turn only after asker ends turn
set-flag-eliminationClient -> Server{ flagCode: string, eliminated: boolean }Set a player's private eliminated state for a flag; broadcast "board-updated" to that player only
end-turnClient -> Server{}Switch turn, notify both players
make-guessClient -> Server{ guessedFlagCode: string }Compare to opponent secret -> emit win/lose + game-over
chat-messageClient -> Server{ text: string }Broadcast to both (for non-question chat if desired)
new-gameClient -> Server{}Reset secrets, boards, score

5. FRONTEND COMPONENT BREAKDOWN

  • Header: Logo + Player labels (RED / BLUE) + Score (Championship mode)
  • Secret Slot: Shows YOUR mystery flag (never opponent's)
  • Personal Grid: 24 clickable flag cards (6x4) with eliminate animation
  • Live Chat: Real-time messages + YES/NO quick buttons (opponent only)
  • Turn Indicator: Big glowing "YOUR TURN!" banner
  • Guess Modal: Dropdown of remaining flags + Submit (instant loss on wrong guess)
  • Mobile Responsive: Works on phones (portrait mode)
